## 🎯 最常用的3个模块

### 模块1：💰 价格咨询（ID: 5）
```
🎯 场景：客户问价格、报价、费用
🔑 触发词：价格、报价、多少钱、成本

📝 默认内容：
  ✓ 3个价格方案（基础/标准/旗舰）
  ✓ 如何选择指南
  ✓ 销售代表信息
  ✓ 详情链接

✏️ 怎么改：
  1. 点击编辑
  2. 修改价格体系（改为你的产品）
  3. 修改销售代表名字
  4. 保存

示例改法：
原：基础版 {price_basic}元
改：小型方案 3999元（最畅销⭐）
```

### 模块2：🔧 售后维修（ID: 6）
```
🎯 场景：客户报修、故障、维修
🔑 触发词：维修、报修、坏了、故障、修理

📝 默认内容：
  ✓ 自动创建工单
  ✓ 技术员信息
  ✓ 预约时间
  ✓ 承诺文案
  ✓ 查询链接

✏️ 怎么改：
  1. 点击编辑
  2. 修改"我们的技术员"信息
  3. 修改承诺（例如上门时间）
  4. 保存

示例改法：
原：技术员{technician}将准时上门
改：技术员李师傅（经验10年）将2小时内上门
```

### 模块3：🎁 活动推广（ID: 12）
```
🎯 场景：推送营销活动、优惠信息
🔑 触发词：活动、促销、优惠、打折、特价

📝 默认内容：
  ✓ 活动标题
  ✓ 优惠幅度
  ✓ 优惠券码
  ✓ 3个商品推荐
  ✓ 分享赚奖励

✏️ 怎么改：
  1. 点击编辑
  2. 改活动名称（例如"双11大促"）
  3. 改优惠幅度（例如"20%折扣"）
  4. 改优惠券码
  5. 改商品信息
  6. 保存

示例改法：
原：优惠力度：{discount_percentage}%
改：优惠力度：满1000减100
```

## 🚫 3个禁用操作

### ❌ 删除系统预留模块
```
如果点击删除系统模块，会看到：

"此模块为系统预留模块，不可删除！
💡 您可以通过启用/禁用开关来关闭此模块"

解决方法：
→ 使用"启用/禁用"开关禁用（不会删除）
→ 需要时可以随时重新启用
```

### ❌ 误删了数据
```
如果发生了：
1. 编辑到一半，不小心删除了内容
2. 关闭了对话框，数据没保存

解决方法：
→ 重新打开该模块的编辑
→ 原数据仍在（系统预留模块的内容不会丢失）
→ 可以重新开始编辑
```

### ❌ 看不到预留模块
```
如果发生了：
1. 打开AI回复模板，但看不到8个模块
2. 只看到之前的2个

解决方法：
→ 刷新页面（F5）
→ 清空浏览器缓存
→ 检查后端服务是否运行
```

## 💪 3个常见改法

### 改法1：替换品牌/公司名
```
搜索和替换（Ctrl+H）：

搜索：【原公司名】
替换为：【你的公司名】

搜索：{sales_representative_name}
替换为：你的销售代表名字
```

### 改法2：调整价格体系
```
在模板中找到价格部分：

原：
基础版：{price_basic}元
标准版：{price_standard}元
旗舰版：{price_premium}元

改为：
普通用户版：1999元
商用版：4999元
企业定制版：按需报价
```
