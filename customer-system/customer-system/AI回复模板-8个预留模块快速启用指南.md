# 🤖 AI回复模板 - 8个预留模块使用指南

## 📝 概述

系统现在为您预留了 **8个完整的AI回复模板**，涵盖所有常见业务场景。这些模板是**系统预留模块**，不可删除（防止误删除），但您可以随时编辑内容来适应您的业务需求。

---

## 📦 8个预留模块一览

### 1. 💰 价格咨询AI回复
- **ID**: 5
- **应用场景**: 客户询问价格、报价、费用
- **触发关键词**: 价格、报价、多少钱、费用、成本
- **功能**: 自动返回价格体系和选择建议
- **可用变量**: {product}、{price_basic}、{price_standard}、{price_premium} 等

### 2. 🔧 售后维修AI回复
- **ID**: 6
- **应用场景**: 客户报修、设备故障
- **触发关键词**: 维修、报修、坏了、故障、修理
- **功能**: 自动创建工单，记录技术员信息和预约时间
- **可用变量**: {product}、{issue}、{technician}、{ticket_id} 等

### 3. 📦 订单确认AI回复
- **ID**: 7
- **应用场景**: 订单确认后自动通知
- **触发关键词**: 订单、确认、已下单、订单号
- **功能**: 显示订单详情和支付链接
- **可用变量**: {customer_name}、{order_id}、{amount} 等

### 4. ❓ 常见问题AI回复
- **ID**: 8
- **应用场景**: 回答产品相关常见问题
- **触发关键词**: 怎么样、如何、使用方法、功能
- **功能**: 自动返回常见问答和技术支持方式
- **可用变量**: {product}、{sales_representative_name}、{baseUrl} 等

### 5. 📊 数据统计AI回复
- **ID**: 9
- **应用场景**: 提供销售、服务、库存数据统计
- **触发关键词**: 统计、数据、报表、分析
- **功能**: 返回业绩统计和可视化链接
- **可用变量**: {today_orders}、{month_orders}、{growth_rate} 等

### 6. ⏰ 预约提醒AI回复
- **ID**: 10
- **应用场景**: 发送上门服务预约提醒
- **触发关键词**: 预约、时间、什么时候、排期
- **功能**: 显示预约信息和取消链接
- **可用变量**: {appointment_date}、{appointment_time}、{technician} 等

### 7. 💬 咨询转接AI回复
- **ID**: 11
- **应用场景**: 转接到人工客服处理
- **触发关键词**: 转接、人工、客服、销售代表
- **功能**: 显示负责人信息和联系方式
- **可用变量**: {assigned_person_name}、{assigned_person_phone} 等

### 8. 🎁 活动推广AI回复
- **ID**: 12
- **应用场景**: 推送营销活动和优惠信息
- **触发关键词**: 活动、促销、优惠、打折、特价
- **功能**: 显示优惠券码和购买链接
- **可用变量**: {discount_percentage}、{coupon_code}、{activity_name} 等

## 🎯 如何使用这些模块

### 第1步：打开AI回复模板管理
```
侧边栏 → 消息模板管理 → 🤖 AI回复模板 标签页
```

### 第2步：查看预留模块
```
您将看到8个预留模块（都标记为"系统预留"）：

| ID | 模块名称 | 状态 | 操作 |
|-----|---------|------|------|
| 5 | 💰 价格咨询AI回复 | 启用 | 编辑 预览 🔒禁用 |
| 6 | 🔧 售后维修AI回复 | 启用 | 编辑 预览 🔒禁用 |
| 7 | 📦 订单确认AI回复 | 启用 | 编辑 预览 🔒禁用 |
| ... | ... | ... | ... |
```

### 第3步：编辑模板内容
1. **点击"编辑"按钮** → 打开编辑对话框
2. **修改内容** → 根据您的业务需求调整文案
3. **调整变量** → 系统已经预设了所有需要的变量，您也可以自行添加
4. **保存模板** → 点击"保存模板"按钮

示例：编辑价格咨询模板
```
原内容：
"您好！关于{product}的价格，我们提供以下方案：
基础版：{price_basic}元
标准版：{price_standard}元  
旗舰版：{price_premium}元"

编辑为：
"您好！{customer_name}，感谢关注我们的{product}！

我们为您准备了三个价格方案：

🔷 基础版 - {price_basic}元
   适合个人用户和小企业
   包含：基础功能、免费更新

🔶 标准版 - {price_standard}元（推荐⭐⭐⭐）
   适合中小企业
   包含：完整功能、优先支持、定制化

🔴 旗舰版 - {price_premium}元
   适合大企业和定制需求
   包含：全部功能、24小时客服、专属账户管理

需要帮助选择？我是{sales_representative_name}，请点击了解详情：
{baseUrl}/product/{product}"
```

### 第4步：启用/禁用模块
```
不想使用某个模块？不用删除，只需：
1. 点击该模块的"启用/禁用"开关
2. 模块会被禁用，但不会被删除
3. 以后可以随时重新启用
```

## ⚠️ 重要提示

### 系统预留模块的特点
```
✅ 不可删除 - 防止误删除数据
✅ 可编辑 - 所有内容都可以修改
✅ 可启用/禁用 - 不用的模块可以关闭
✅ 支持自定义变量 - 可以添加自己的变量
✅ 预设了最佳实践 - 模板内容已经优化
```

### 删除系统预留模块会怎样？
```
如果您点击了🔒禁用按钮，会弹出提示：

"此模块为系统预留模块，不可删除！

💡 您可以通过"启用/禁用"开关来关闭此模块，而不会丢失数据。

模块名称：💰 价格咨询AI回复"

这意味着：
❌ 您无法删除预留模块
✅ 但可以随时通过启用/禁用开关来控制是否使用
✅ 编辑过的内容会被保留
```

## 🔧 常见操作

### Q1: 我想新建一个自定义模板，不用预留模块
```
✅ 可以的！系统预留模块只是默认提供的，不限制自定义。

操作步骤：
1. 点击"新建模板"按钮
2. 选择模板类型为"AI回复模板"
3. 填写您的内容
4. 保存

这样创建的模板：
✅ 可以编辑
✅ 可以删除（与预留模块不同）
❌ 不会被标记为"系统预留"
```

### Q2: 我不想用某个模块，但又不想永久删除
```
✅ 使用启用/禁用功能！

操作步骤：
1. 找到该模块
2. 点击"启用/禁用"开关
3. 模块会变为禁用状态，不会在AI回复中被触发
4. 需要时，随时可以重新启用
```

### Q3: 我想完全复制一个模块，进行大幅修改
```
✅ 操作步骤：
1. 编辑原模块，复制所有内容
2. 点击"新建模板"
3. 粘贴内容并修改
4. 另存为新的模板（例如"价格咨询-高端版"）

这样：
✅ 原模板保留不变（系统预留）
✅ 新模板保留您的定制（可删除）
✅ 可以同时使用两个版本
```

### Q4: 如何知道哪个模板被触发了？
```
在"AI对话功能总结"文档中查看每个模块的触发关键词。

例如：
• 价格咨询模块：触发词 = "价格、报价、多少钱、费用、成本"
• 售后维修模块：触发词 = "维修、报修、坏了、故障、修理"

系统会自动根据客户的消息内容，匹配对应的模块。
```
