# 本地开发快速启动指南

## 🚀 一键启动（推荐）

直接运行启动脚本：

```powershell
.\start-dev.ps1
```

这将自动启动前端和后端服务，打开两个独立的窗口。

---

## 📋 环境要求

- **Python 3.11+**
- **Node.js 18+**
- **Redis**（可选，不影响基本功能）

---

## 🔧 分步启动

### 1️⃣ 启动后端

```powershell
.\start-backend.ps1
```

后端将运行在：http://localhost:8000
API文档：http://localhost:8000/docs

### 2️⃣ 启动前端

打开新的终端窗口：

```powershell
.\start-frontend.ps1
```

前端将运行在：http://localhost:3000

---

## 💡 开发说明

### 数据库
- 开发环境使用 **SQLite**（自动创建 `customer_system.db`）
- 无需安装 PostgreSQL
- 数据库文件位置：`backend/customer_system.db`

### 热重载
- 后端：自动重载（修改代码后自动重启）
- 前端：自动热更新（修改代码后浏览器自动刷新）

### 停止服务
关闭对应的 PowerShell 窗口即可

---

## 🐛 常见问题

### Python 虚拟环境激活失败
运行以下命令允许脚本执行：
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

### 端口占用
如果 8000 或 3000 端口被占用，修改脚本中的端口号。

### 缺少依赖
确保已安装 Python 和 Node.js，然后脚本会自动安装其他依赖。

---

## 📦 完整安装步骤（首次运行）

1. **安装 Python**：https://www.python.org/downloads/
2. **安装 Node.js**：https://nodejs.org/
3. **运行启动脚本**：`.\start-dev.ps1`

就这么简单！
