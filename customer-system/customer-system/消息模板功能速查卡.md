# 📋 消息模板功能速查卡

## 🎯 快速决策表

```
需要配置消息模板？请先确定：

1️⃣ 选择模板类型
   ├─ 📢 群机器人    → 支持「实时」+「定时」
   ├─ 🤖 AI回复      → 支持「实时」+「定时」
   ├─ 🏢 企业微信    → 支持「实时」+「定时」
   ├─ 💬 公众号      → 支持「实时」+「定时」
   ├─ 📱 短信        → 无需配置推送模式
   └─ 📧 邮件        → 无需配置推送模式

2️⃣ 选择推送模式
   ├─ ⚡ 实时推送   → 关键词触发，立即响应
   └─ ⏰ 定时推送   → 按计划发送，定期推送

3️⃣ 配置必填项
   实时推送必填：
   ✅ 触发关键词
   ✅ 发送对象
   ✅ 模板内容
   
   定时推送必填：
   ✅ 发送时间
   ✅ 发送对象
   ✅ 模板内容
```

---

## ⚡ 实时推送 vs ⏰ 定时推送

| 对比项 | 实时推送 | 定时推送 |
|--------|---------|---------|
| **触发方式** | 关键词匹配 | 定时计划 |
| **响应速度** | 秒级响应 | 按计划执行 |
| **适用场景** | 客服问答、自动回复 | 通知、日报、提醒 |
| **配置重点** | 触发关键词 | 发送时间、重复周期 |
| **典型示例** | "价格" → 自动回复 | 每天9点发工作提醒 |

## 🤖 AI回复模板速查

### 关键词匹配逻辑
```
触发关键词: "价格,报价,咨询"

客户消息包含任一关键词即触发：
✅ "请问价格多少？"        → 触发（包含"价格"）
✅ "能给我报价吗？"        → 触发（包含"报价"）
✅ "我想咨询一下"          → 触发（包含"咨询"）
❌ "你好"                  → 不触发
❌ "谢谢"                  → 不触发
```

### 推荐配置
```yaml
实时响应场景:
- 产品咨询 → 关键词: 价格,产品,咨询
- 售后问题 → 关键词: 坏了,故障,维修
- 订单查询 → 关键词: 订单,查询,进度

定时推送场景:
- FAQ定期推送 → 每周五更新
- 知识库提醒 → 每月1日推送
```

## 🚀 快速操作指南

### 创建实时推送模板（3步）
```
1. 点击「+ 新建模板」
2. 配置基本信息：
   - 模板名称
   - 推送模式: ⚡ 实时推送
   - 触发关键词: 价格,报价
   - 发送对象: 客户服务群
3. 填写模板内容 → 保存
```

### 创建定时推送模板（4步）
```
1. 点击「+ 新建模板」
2. 配置基本信息：
   - 模板名称
   - 推送模式: ⏰ 定时推送
3. 配置定时：
   - 发送时间: 09:00
   - 重复周期: 每天
   - 发送对象: 内部工作群
4. 填写模板内容 → 保存
```

### 测试模板（2步）
```
1. 在模板列表找到目标模板
2. 点击「测试发送」→ 选择测试对象 → 确认发送
```

## ⚠️ 常见错误速查

| 错误提示 | 原因 | 解决方法 |
|---------|------|---------|
| "实时推送必须填写触发关键词" | 选择实时推送但未填关键词 | 填写触发关键词 |
| "定时推送必须选择发送时间" | 选择定时推送但未选时间 | 选择发送时间 |
| "发送时间必须大于当前时间" | 选择的时间已过期 | 重新选择未来时间 |
| "请选择发送对象" | 未选择目标群聊/成员 | 至少选择一个对象 |
| "API密钥错误" | AI模型配置错误 | 检查AI模型配置 |
| "公众号定时时间不得超过7天" | 公众号限制 | 调整时间在7天内 |
